Local API REST

Cutre versión local de un servidor REST para realizar pruebas.

Funciones de la API

POST /test/:id

Devuelve un fichero con la siguiente información:

  • Cabeceras de la solicitud HTTP
  • Query String de la solicitud HTTP, es decir, todos los pares nombre - valor que hay en la URI después del símbolo de la interrogación.
  • Parámetros REST, solo acepta un parámetro que es el valor :id de la URI
